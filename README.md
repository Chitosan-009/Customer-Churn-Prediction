📊 Customer Churn Prediction

本项目基于 Kaggle Telecom Churn 数据集，通过端到端的机器学习流程构建客户流失预测模型，重点不在于单一模型效果，而在于构建一个 可解释、可落地、可复现、可指导运营决策的客户流失预测与风险预警体系。
🎯 项目背景

在电信行业中，客户流失直接影响收入稳定性与获客成本。运营方普遍面临以下问题：

新用户获取成本持续上升；

客户流失率长期处于较高水平；

用户对资费高度敏感，市场竞争激烈；

套餐设计与用户真实使用行为存在不匹配。

本项目希望通过数据分析与建模，解决以下业务问题：

提前识别未来一段时间内的高流失风险用户；

找到导致客户流失的关键驱动因素；

为运营与客服团队提供量化、可执行的干预依据；

辅助提升用户生命周期价值（CLV）。
📦 数据来源与内容

数据来源：Kaggle Telecom Churn Dataset

训练集规模：4,250 条用户记录

特征构成：19 个特征 + 目标变量 churn

主要变量包括：

用户基本信息：state、area_code

套餐信息：international_plan、voice_mail_plan

通话行为特征：日 / 晚 / 夜通话分钟数、费用、通话次数

服务行为特征：customer_service_calls（客服呼叫次数）
